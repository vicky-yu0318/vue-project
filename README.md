## 專案說明
* 網站主要使用 Vue.js 及 Vue Cli 架設而成
* 加入 ESLint 撰寫風格
* 透過 mitt 傳遞跨元件資料
* 開發後台管理系統

### 前台
* 電商展示：首頁、商品列表、商品介紹等
* 購物車：刪除、修改商品數量
* 優惠券
* 結帳

### 後台
* 管理者登入
* 商品管理
* 優惠券管理
* 訂單管理
* 文章管理
